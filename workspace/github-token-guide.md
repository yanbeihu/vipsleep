# GitHub Token 生成指南 - 3 分钟搞定

## 📋 按照以下步骤操作：

### 第 1 步：访问 GitHub 设置
打开这个链接：
https://github.com/settings/tokens

### 第 2 步：创建新 Token
1. 点击右上角的 **"Generate new token"** → **"Generate new token (classic)"**

### 第 3 步：设置 Token
1. **Note**：输入 `VIPSLEEP-Deploy`
2. **Expiration**：选择 `90 days`（90天有效期）
3. **Select scopes**：勾选 ✅ **`repo`** （这个最重要！）
4. 点击底部的 **"Generate token"** 按钮

### 第 4 步：复制 Token
⚠️ **重要**：Token 只显示一次，请立即复制！
看起来像这样：`ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

### 第 5 步：发给我
把复制的 Token 发给我，格式如：
```
Token: ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
```

---

## 🔐 安全说明
- 这个 Token 只用于部署网站，非常安全
- 90天后会自动过期
- 您随时可以在 GitHub 设置里撤销
